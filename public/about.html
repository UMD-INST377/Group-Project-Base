About Our Dataset

Our application is based on a dataset from the National Basketball Association (NBA). The database that application will be using was developed in INST327 Database Design and Modeling, by Quest and his team. The database was built using sources such as https://stats.nba.com/, which is the official statistics page offered by the NBA. Another source is http://www.espn.com/nba/salaries, which provides player salary information which can not be found on the NBAâ€™s website.

Our Goal

This application will solve the problem by giving one central location for users to go to access information and facts about NBA teams and players, therefore taking out the need to visit multiple sites to retrieve the information such players stats, team info, team staff, etc.

This web application was created by Team 12 for Spring 2021 INST377.
