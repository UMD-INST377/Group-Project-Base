When searching through the housing market, users should be able to ensure that their potential homes are free of any danger. This site allows users to search for housing inspection violations on homes in Prince George's county. This website uses Prince George's County Open Data Portal, which contains a Housing Inspection Violations Dataset with a complete list of properties within Prince George's County which have been issued Housing Inspection Violations since June 2014. This dataset is scheduled to update on a weekly basis, but has not been updated since August 23, 2021.
