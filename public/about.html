About

This is a project designed to combat existing difficulties in finding available data on earthquakes.

We have found that many solutions for making earthquake data available to the general public lacks intuitive UI, so we are seeking to provide an easier way to retrieve and work with data.

This is a project for Fall 2021 semester of INST377 - Dynamic Web Applications at the University of Maryland. You can find the project's codebase on GitHub by clicking on the image.
