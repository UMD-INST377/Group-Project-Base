About Us
Mission
The purpose of our project is to help UMD students, staff, and faculty members find resteraunts around campus efficinetly.
Exisiting Issue
A lot of people throughout campus are facing issues due to a lack of information. People are unaware of what restaurant offers what food or what time the restaurant closes. Some people might also wish to know how much calories a particular food has or how much sodium it has.
Our Solution
By creating this website and leveraging the existing data from the database, we can give people the answers to their existing questions. People will be able to find what restaurant offers their favorite food, what time it closes, and how does that food exactly affect you (in matters of calories or sodium). Our website will make your search far more efficient than what it is currently.

Developer Manual
Installing application
1. Clone this repository through Github or Terminal
2. Open the repository in a Terminal application
3. Install npm and run it
4. On Terminal it should now say Listening on: http//localhost:3000
Running the application
1. Open a Terminal window up and make sure you are on the currect project directory
2. Install npm and run it
3. On Terminal it should now say Listening on: http//localhost:3000 and you would go to a web browser and type in the url http//localhost:3000
Running tests for the software
We have not set up any tests but you are able to run your own tests and use npm
Server APIs
/api - will indicate you are in UMD Dining API. GET - will get the data from written query
/api.POST - posts the database data based the on the request body
/api.PUT - updates data
/api.DELETE - deletes data from the database
