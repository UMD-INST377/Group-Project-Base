Developer Manual How to install your application and all dependencies Clone the chose repository through the Terminal or the GitHub Desktop Application Open the repository within VSCode and open a terminal Within the terminal type npm install The application should be ready to be used How to run your application on a server Open the repository and a terminal Type npm start within the terminal If working correctly, go to http://localhost:3000/ How to run any tests you have written for your software Open two seperate terminals with VSCode Type npm start within one of the terminals Type npm test within the other terminal API Information Included in the repository is a list of API routes which correlate to their respective tables in the database. They include: collisionType crashInformation driverDemographics roadConditions culpability Each endpoint contains GET, POST, PUT, and DELETE methods to allow for data manipulation. The methods and their uses are detailed below. GET - Retrieves data from the server. POST - Sends data to the server allowing for the creation of new resources. PUT - Sends data to the server allowing for existing records to be updated. DELETE - Deletes a resource from the server.
