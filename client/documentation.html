User Manual

Home

Use the home page to navigate our web application. The main navigation will take you to the data graphs, new record, and about pages. Users can also download the SQL data or navigate to our GitHub repository for more information.

Data Graphs

Here the user can see the data visualized in real-time as the database is updated.

New Record

Use the forms to add new data to the database. First select from the dropdown menu of what data you would like to enter. Fill in all the required fields then click submit. The data will automatically populate into the database.
